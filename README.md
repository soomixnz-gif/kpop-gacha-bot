# K-Pop Gacha Bot

## Overview

The K-Pop Gacha Bot is a fun and interactive application designed to simulate the experience of collecting K-Pop idol cards. Users can engage in gacha mechanics similar to those found in popular mobile games, where they can randomly acquire various K-Pop idols, enhance their collections, and participate in events to earn exclusive items.

## Features
- **Gacha Mechanics**: Users can draw random cards from different sets, each containing various K-Pop idols.
- **Collection Management**: Users can view and manage their collections, track which idols they have obtained, and see which ones are still missing.
- **Events and Rewards**: Participate in special events that offer unique cards and bonuses.
- **User Friendly Interface**: An intuitive UI enables easy navigation and interaction with the bot.

## Getting Started
To get started with the K-Pop Gacha Bot, follow these instructions:

1. **Clone the Repository**:  
   `git clone https://github.com/soomixnz-gif/kpop-gacha-bot.git`

2. **Install Dependencies**:  
   `npm install`

3. **Run the Bot**:  
   `node index.js`
